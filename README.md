# OTT DRM Play

Это веб-приложение для просмотра потокового видео с поддержкой DRM (Digital Rights Management).

## Особенности

- Поддержка потокового видео
- DRM-защита контента
- Адаптивный дизайн для различных устройств
- Поддержка различных форматов видео

## Технологии

- HTML5
- JavaScript
- MPEG-TS потоковое видео
- DRM технологии

## Структура проекта

```
src/ott.drm-play.com/
├── index.html          # Главная страница
├── manifest.webmanifest # Web App Manifest
├── robots.txt          # Файл для поисковых роботов
├── fonts/              # Шрифты
├── js/                 # JavaScript файлы
│   ├── jq-1.11.1.min.js # jQuery
│   └── mpegts.js      # MPEG-TS потоковый плеер
└── pc/                 # ПК-версия
    ├── stb.js         # STB логика
    └── stbPlayer.js   # STB плеер
```

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone <repository-url>
   ```

2. Перейдите в директорию проекта:
   ```bash
   cd ott-app
   ```

3. Запустите локальный сервер:
   ```bash
   # Например, с помощью Python
   python -m http.server 8000
   ```

4. Откройте в браузере:
   ```
   http://localhost:8000/src/ott.drm-play.com/
   ```

## Использование

Приложение автоматически определяет тип устройства и загружает соответствующую версию интерфейса. Для просмотра контента необходимо иметь соответствующие DRM-лицензии.
