# OTT DRM Play

Более "открытая" версия OTT-Play от Alex - веб-приложение для просмотра потокового видео.

## Особенности

- Просмотр потокового видео
- Поддержка MPEG-TS формата
- Адаптивный дизайн для различных устройств
- Полноэкранный режим отображения
- Информационные OSD элементы (время, каналы, прогресс)
- Поддержка управления громкостью
- Возможность просмотра в режиме PiP (Picture-in-Picture)

## Технологии

- HTML5
- JavaScript (jQuery 1.11.1)
- MPEG-TS потоковый плеер
- Web App Manifest
- CSS3 стилизация

## Установка и запуск

### Вариант 1: Запуск как десктопного приложения (Electron)

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите приложение:
   ```bash
   npm start
   ```

### Компиляция десктопного приложения

Для создания установочных пакетов десктопного приложения:

1. Установите зависимости (если еще не установлены):
   ```bash
   npm install
   ```

2. Соберите приложение:
   ```bash
   # Сборка без публикации
   npm run dist
   
   # Или полная сборка с публикацией
   npm run build
   ```

3. Собранные пакеты будут находиться в папке `dist/`:
   - `ott-app-1.1.0.AppImage` - для Linux (универсальный пакет)
   - `ott-app-1.1.0.deb` - для Debian/Ubuntu систем

### Требования для компиляции

- Node.js (версия 14 или выше)
- npm (входит в состав Node.js)
- Для Linux: установленные системные зависимости:
  - `libgconf-2-4`
  - `libappindicator3-1`
  - `libxtst6`
  - `libnss3`
  - `libasound2`

### Особенности сборки

- Приложение собирается с использованием Electron 27.0.0
- Используется electron-builder для создания установочных пакетов
- Поддерживаются форматы AppImage и DEB для Linux
- Приложение упаковывается в ASAR архив для оптимизации
- Автоматически включаются все необходимые файлы, кроме избыточных (документация, тесты, исходные карты)

## Использование

Приложение предназначено для просмотра потокового видео. Интерфейс включает:
- Информационную панель с отображением текущего канала и программы
- Прогресс-бар просмотра
- Управление громкостью
- Режим буферизации
- Возможность просмотра в полноэкранном режиме
